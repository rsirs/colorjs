# ColorJS Documentation

Welcome to the ColorJS library documentation.

ColorJS is a TypeScript library for color manipulation and conversion, providing comprehensive color utilities and calculations across various color spaces.

## Getting Started

- [Installation](../README.md#installation)
- [Usage](../README.md#usage)

## API Reference

- [Color Class](./colorjs.color.md) - Main color manipulation class
- [ColorUtil Class](./colorjs.colorutil.md) - Color utility functions

## Features

- Color class with RGB input
- Support for multiple color spaces: HSV, HSL, HSI
- Color utility functions for conversions and calculations
- TypeScript support with full type definitions
